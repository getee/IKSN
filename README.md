## IKSN
互联网知识共享平台，主打博客+资源模块。

基于IDEA2017+Tomcat7.0+JDK1.8开发。

## 功能实现
- [x] 通过手机验证码注册
- [x] 编写博客 ueditor插件
- [x] 站内即时通信 web socket
- [x] 用户间关注,评论,私信,举报
- [x] 拼音,模糊 检索
- [x] 资源特征码提取查重，上传下载
- [x] 管理员处理举报博客，资源
